## 安装方法

每个 Skill 是一个 `SKILL.md` 文件，复制到 Claude 的 skills 目录即可：

```bash
# macOS / Linux
cp flow2-graph-prompt/SKILL.md ~/.claude/skills/flow2-graph-prompt/SKILL.md
```

然后在 Claude Code 中直接调用：

```
/flow2-graph-prompt
```
