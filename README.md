## How to install

Each skill is a single `SKILL.md` file. Copy it into your Claude skills directory:

```bash
# macOS / Linux
cp flow2-graph-prompt/SKILL.md ~/.claude/skills/flow2-graph-prompt/SKILL.md
```

Then invoke it in Claude Code:

```
/flow2-graph-prompt
```
